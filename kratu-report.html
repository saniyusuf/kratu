Kratu · Status & Roadmap · 9 Jul 2026

Where Kratu stands A voice-first, offline literacy tutor for Hausa speakers — what works today, and where it can go.

Try it
The alphabet lessons run locally at http://localhost:8791/kratu-lessons.html. Everything below refers to that working prototype — no accounts, no API keys, and after the model loads once, no internet.

01What works today

A complete alphabet lesson loop, driven by the learner's voice, running entirely on-device.

  • LIVE
    Offline speech recognition (Vosk)
    An on-device recogniser (WebAssembly) hears the learner say letters — no cloud, works with no internet. This replaced the flaky browser/Google recogniser and is the big reliability win.
  • LIVE
    Real Hausa tutor voice
    Every prompt, praise and reminder is spoken in Hausa (pre-rendered, embedded in the page). Letters are said the English way.
  • LIVE
    Staged alphabet board (A–Z)
    The whole alphabet as a board, split into 7 groups of ~4. Groups unlock in order; finished ones turn gold. Progress is saved on the device.
  • LIVE
    Teach → recite → mix, per group
    Each letter taught (said twice), then recited in order, then shuffled — nothing advances until the learner says them. Right → kudos, wrong → retry, two misses → the tutor reminds by saying it.
  • LIVE
    Two tests
    A main test (recite A→Z) and a final sequence-completion test (“O P Q R … what's next?”).
  • LIVE
    Gender-aware Hausa
    Uses the correct ka/ki, -nka/-nki forms in both audio and text based on the learner's gender.
  • LIVE
    Classroom animations
    Background colour + a raised hand signal whose turn it is to talk (tutor = indigo, your turn = gold hand-raise, you speaking = green).
  • LIVE
    Scoring
    Per-letter attempts, correct count, and what was said when wrong — shown in a results table.
  • LIVE
    Onboarding (name + gender)
    Gender select and spoken/typed name capture, greeting the learner by name — and it flows straight into the lessons.

02How it's built

Recognition

Vosk small English model (~40 MB), running in-browser via WebAssembly, constrained to the current group's letters for accuracy. Fully offline.

Voice

Pre-rendered Hausa + English clips embedded as audio data — instant, offline, consistent. The “pre-authored curriculum” model.

App

A single self-contained HTML page. No backend, no API keys. Progress in the browser's local storage.

Deployment fit

Serves from local files — exactly how the offline learning centre (Starlink-optional) will run on its own tablets.

03Honest notes & current limits

Note
The working files were briefly lost when a temporary folder was cleared, and have been rebuilt and moved to a permanent location (~/Documents/kratu/) so this can't happen again. The name bank was also expanded from 45 to 117 names in the process.
  • Letter recognition isn't perfect. Isolated letters are acoustically hard (B/C/D/E all sound alike). Vosk + a per-group grammar + a lenient nearest-letter matcher makes it good, not flawless; a skip button means it never dead-ends.
  • One Hausa voice. The free voice sounds female. A true male voice (and richer intonation) needs a production TTS — the app is already structured for it.
  • Teaching pace is deliberate (each letter said twice) — tunable.
  • Browser-only for now. Packaging it as a tablet app for the centre is a straightforward next step.

04Where it can go — possible features

Near term

Finish the alphabet track

  • Lowercase pass — teach a b c d… after the capitals, and matching cases.
  • Facilitator view — a room dashboard: who's on which group, who's struggling, per-learner scores.
  • Better letter accuracy — tune the grammar/matcher per group; optional male voice.
  • Adaptive review — resurface the letters a learner keeps missing.
Mid term

From letters to reading

  • Phonics & blending — letter sounds, then simple CVC words (cat, dog) with pictures.
  • Sight words & short sentences — read-along and read-back.
  • Numbers & basic maths — the exact same teach→say-back→test loop, with unambiguous answers (uku → 3).
  • Writing / tracing — trace the letter on the tablet, not just say it.
Bigger vision

The real product

  • Pronunciation scoring — not just “right letter” but how well it was said (the original goal).
  • Free-conversation mode — talk to it in Hausa about anything, English-pivot under the hood.
  • Progress over time & sync — a learner profile that follows the person across tablets, syncing when Starlink is up.
  • More languages — the same engine for other Nigerian languages.
  • Photo/AI login for non-readers, and real recorded voice talent for warmth.
